### 项目结构介绍

```
AS
│   README.md
│  
└───ASProj 主项目
│   
└───hi-ui ui组件库
│   
└───hi-library library组件库
│   
└───hi-demo 架构师demo中心
```

### 如何运行

1. 本仓库项目可独立运行
2. 将[hi-library](https://git.imooc.com/class-85/hi-library.git) clone下来
3. 用[Android Studio 4.1](https://developer.android.com/studio/preview)及以上版本打开项目运行
